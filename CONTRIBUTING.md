# Contributing

UACC v0.2 is a public draft. Contributions should focus on clarity, implementability, evidence quality, crosswalk accuracy, and safe public positioning.

Use GitHub Issues or Discussions for feedback on control wording, crosswalk confidence labels, evidence expectations, governance-as-code validation, sector parameter packs, proposed overlays, or implementation experience.

## Contribution guidelines

- Do not add compliance guarantees or certification claims.
- Preserve crosswalk confidence labels: Direct, Partial, Analogical, Informative.
- Use normative/informative labels for control text.
- Use `shall` for mandatory requirements and `should` for recommendations.
- Do not add third-party copyrighted material unless licensing is clear.
- Prefer issue discussions for new sector packs, new overlays, or major scope expansion.
